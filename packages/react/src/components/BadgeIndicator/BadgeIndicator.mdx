## Overview

`BadgeIndicator`

```jsx
import { BadgeIndicator as unstable__BadgeIndicator } from '@carbon/react';
import { IconButton } from '@carbon/react';


function ExampleComponent() {
  return (
    <IconButton kind="ghost" label="Icon Description with 1234 notifications">
      <Notification />
      <BadgeIndicator count={1234}></BadgeIndicator>
    </IconButton>
  );
}
```

## Count

You can set the count to appear to indicate on the badge. Without a count the indicator will still appear, but without a number.
